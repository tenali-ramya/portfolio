Campus Placement Prediction Using Machine Learning

This project focuses on predicting campus placement outcomes using machine learning techniques applied to student academic, technical, and personal data. The objective is to help students and placement cells understand the factors that influence employability and placement success. The project involves data collection, cleaning, and exploratory data analysis to identify patterns and trends. Multiple machine learning classification models are trained and evaluated to predict whether a student is likely to be placed.

Automatic Railway Gate Controlling System Using IR Sensor

This project focuses on enhancing safety at unmanned railway level crossings by automating gate operations using IR sensors. The system detects the arrival and departure of trains and automatically controls the opening and closing of railway gates.
